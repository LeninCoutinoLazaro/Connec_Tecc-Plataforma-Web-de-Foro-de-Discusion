# Connec_Tecc — Simulación de Estadísticas de Interacción en un Foro

## Descripción

Connec_Tecc es una aplicación web desarrollada con el propósito de simular el comportamiento estadístico de una plataforma de foro de discusión. El sistema permite visualizar información almacenada en una base de datos mediante gráficas dinámicas que representan distintos tipos de interacción dentro de la plataforma.

La aplicación fue creada como un proyecto académico con el objetivo de demostrar la integración entre una aplicación web desarrollada en ASP.NET y una base de datos relacional, así como la representación gráfica de información utilizando herramientas de visualización de datos.

En lugar de implementar un foro completamente funcional, el sistema se enfoca en simular el análisis estadístico de la actividad dentro de una comunidad digital, mostrando la distribución de hilos de discusión, categorías y subcategorías mediante gráficos interactivos.

---

## Tecnologías utilizadas

El desarrollo de la aplicación se realizó utilizando las siguientes tecnologías:

- Lenguaje de programación: C#
- Framework: ASP.NET Razor Pages
- Base de datos: Microsoft SQL Server
- Gestor de base de datos: SQL Server Management Studio
- Biblioteca de visualización: Chart.js
- Tecnologías de interfaz: HTML, CSS y JavaScript
- Entorno de desarrollo: Visual Studio 2022

---

## Funcionalidades del sistema

La aplicación permite visualizar información estadística almacenada en una base de datos mediante representaciones gráficas. Entre las principales funcionalidades se encuentran:

- Sistema de autenticación con acceso diferenciado para usuarios y administradores.
- Consulta de información almacenada en la base de datos.
- Representación gráfica de datos mediante gráficos de tipo pastel.
- Visualización de estadísticas relacionadas con hilos de discusión.
- Visualización de estadísticas de categorías dentro del sistema.
- Visualización de estadísticas de subcategorías registradas en la base de datos.

Estas representaciones permiten simular el análisis de actividad dentro de una plataforma de discusión digital.

## Estructura del proyecto

El repositorio contiene un archivo comprimido (.rar) que incluye la estructura completa del proyecto, dentro del cual se encuentran:

- Código fuente de la aplicación web.
- Archivos de configuración del proyecto ASP.NET.
- Archivos relacionados con la conexión a la base de datos SQL Server.
- Páginas Razor utilizadas para la interfaz web.
- Scripts utilizados para la generación de gráficas con Chart.js.

## Archivo del proyecto

El proyecto completo se encuentra disponible en el siguiente archivo comprimido:

Connec_Tecc.rar

## Cómo ejecutar el proyecto

Para ejecutar el sistema es necesario realizar los siguientes pasos:

1. Descargar el archivo comprimido disponible en este repositorio.
2. Descomprimir el archivo utilizando un software de descompresión como WinRAR o 7-Zip.
3. Abrir la solución del proyecto desde Visual Studio 2022.
4. Configurar la conexión a la base de datos SQL Server en el archivo de configuración correspondiente.
5. Crear o restaurar la base de datos utilizada por el sistema.
6. Ejecutar la aplicación desde el entorno de desarrollo.

---

## Objetivo del proyecto

El objetivo de este proyecto fue desarrollar una aplicación web capaz de representar visualmente información almacenada en una base de datos mediante el uso de gráficos interactivos. Asimismo, se buscó aplicar conocimientos relacionados con el desarrollo web en el entorno .NET, el manejo de bases de datos relacionales y la visualización de datos dentro de una interfaz web.
